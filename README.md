## ⚙️ 技术细节

### 加密流程
读取图片文件 → ArrayBuffer

生成随机盐(16字节)和IV(12字节)

使用PBKDF2从密码派生AES-GCM密钥

使用AES-GCM加密图片数据

组合: 盐 + IV + 密文

保存为 .enc 文件

打包所有.enc文件到ZIP

### 解密流程
读取ZIP文件，提取.enc文件

从.enc文件分离: 盐(16字节)、IV(12字节)、密文

使用相同密码和盐派生密钥

使用AES-GCM解密数据

恢复原始图片数据

打包所有图片到新ZIP
